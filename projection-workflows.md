# Projection Workflows

## Direct Projection Mode

Use when the phone/tablet is connected to the projector.

Requirements:

- full-screen output
- calibration grid
- brightness controls
- orientation lock
- saved alignment preset

## Export Mode

Use when the user wants to play the mapped content from a computer, media player, USB drive, or projector.

Export package should eventually include:

- final MP4 loop
- alignment guide PNG
- project JSON
- optional mask PNG

## Alignment Principle

The projector must be placed back in the same position and aimed at the same surface.

The alignment guide gives the user a visual reference to line everything up.
