# GlowCast MVP Roadmap

## Phase 1 — Manual Mapping Prototype

Goal: prove the simple workflow.

- photo upload
- manual zones
- include/exclude selected zones
- invert selection
- basic animated overlays
- alignment guide export
- watermark preview

Status: started in this scaffold.

## Phase 2 — Better Mask Editing

- polygon zones
- drag handles
- resize handles
- duplicate zones
- snap-to-grid
- feather edge option
- paint add/remove mask
- undo/redo

## Phase 3 — Video Export

- export MP4 loop
- export alignment image
- export transparent PNG mask
- export project JSON
- preset resolutions: 720p, 1080p, 4K

## Phase 4 — Direct Projection Mode

- full-screen projector output
- calibration grid
- brightness and contrast controls
- save projector alignment preset
- mirror/display guidance

## Phase 5 — AI-Assisted Masking

- detect likely windows/doors/panels
- suggest projection zones
- one-tap include/exclude
- invert suggestions
- manual cleanup tools

## Phase 6 — Template Store

- holiday bundles
- business bundles
- event bundles
- band/stage bundles
- creator marketplace later
